# 智研创新模块中心

## 一、环境要求
- Node.js: v20.20.2 或更高版本
- npm 9.0 或更高版本（或 yarn / pnpm）

## 二、安装
```bash
npm install
```

## 三、运行
```bash
npm run dev
```
浏览器访问：http://localhost:5173

## 四、配置
本项目为纯前端静态项目，通过智能体输出的 JSON 文件导入数据进行展示，无需配置 AppKey 或接口地址。

网站在线访问：[https://lll888666.github.io/zhiyan-agent-web/](https://lll888666.github.io/zhiyan-agent-web/)

源代码仓库：[https://github.com/lll888666/zhiyan-agent-web](https://github.com/lll888666/zhiyan-agent-web/)

## 项目介绍
智研创新模块中心是面向本科生科研训练、论文优化与学术规范学习的智能辅助项目。
本项目由腾讯云智能体开发平台工作流 + 前端交互展示网站共同构成，目标是将传统“只给结果”的科研智能体，升级为**“可解释、可训练、可迁移、可成长”**的教育型科研辅助系统。

### 1. 项目背景
传统科研辅助智能体通常能够完成文献检索、文本润色、基础问答等功能，但在真实使用过程中，仍存在几个明显问题：
- 用户往往只能拿到结果，却不知道“为什么这样改”
- 一次修改很难迁移到下一次写作，缺乏学习闭环
- 风险提示过于分散，不利于快速定位关键问题
- 学术规范教育停留在静态说明层面，缺少互动训练
- 科研训练过程缺少阶段感、成长感与任务感

因此，本项目围绕**“让科研智能体不仅能帮用户完成任务，还能引导用户学会科研表达与规范写作”**这一目标，设计并实现一套创新模块体系。

### 2. 项目定位
本项目适用于以下场景：
- 本科生科研入门训练
- 课程论文与毕业论文写作支持
- 科研写作规范教学
- 论文逻辑审查与风险提示
- 润色结果可解释展示
- 科研任务阶段训练与成长陪伴

**项目整体定位**：一个兼具科研辅助、写作训练、风险识别与成长陪伴的教育型智能体系统。

### 3. 核心创新点
#### 创新点 1：科研成长任务训练站
将科研过程拆解为任务阶段，以“关卡训练”的形式引导用户完成：
- 风险识别训练
- 文献甄别训练
- 实验设计训练
- 学术表达训练

强调科研训练的**“陪伴感”“阶段感”**和**“成长感”**。

#### 创新点 2：失败案例 / 风险知识联动
围绕科研中常见问题建立风险识别框架，例如：
- 结论过度外推
- 对比实验不足
- 指标定义不清
- 图表描述不完整
- 参考文献支撑不足
- 结构缺项
- 学术表达不严谨
- 数据/数值表述可疑

当前前端已支持风险展示与证据卡片呈现，后续可与平台知识库深度联动，实现失败案例检索与规则预警。

#### 创新点 3：可解释润色查看器
将普通的“润色结果”升级为“可学习的润色结果”。
系统不仅输出润色文本，还同步输出：
- 原句
- 修改后句子
- 修改类型
- 修改原因
- 可迁移规则

用户可以从**“看结果”**升级为**“学方法”**。

#### 创新点 4：科研规范训练场
通过学术规范问答、案例辨析、避坑挑战等形式，把抽象规范转化为交互式训练内容。
相比传统的说明文档或知识问答，更强调**“做题训练 + 即时反馈 + 规则内化”**。

### 4. 项目架构总览
#### A. 智能体平台侧（腾讯云智能体开发平台）
负责：
- 用户意图识别
- 工作流调度
- 文献搜索
- 论文润色
- 逻辑审查
- 格式校验
- 图表数据提取
- 结果整合为结构化 JSON

#### B. 前端展示侧（本仓库）
负责：
- 报告展示
- 风险可视化
- 可解释润色对照
- 科研规范训练
- 科研成长任务训练
- 消费平台输出的 JSON 并进行交互展示
